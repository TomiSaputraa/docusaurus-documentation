Operator

Operator adalah simbol khusus yang digunakan untuk melakukan operasi tertentu pada operan. Dart memiliki banyak operator bawaan yang dapat digunakan untuk melakukan berbagai fungsi, misalnya '+' digunakan untuk menambahkan dua operan. Operator dimaksudkan untuk melakukan operasi pada satu atau dua operan.

Tabel Prioritas Operator di Dart:

1. Operator Aritmatika​

Kelas operator ini berisi operator-operator yang digunakan untuk melakukan operasi aritmatika pada operan. Operator-operator ini adalah operator biner, yaitu operator yang bekerja pada dua operan. Operator-operator ini bekerja seperti ini:

Simbol OperatorNama OperatorDeskripsi Operator
+TambahanDigunakan untuk menambahkan dua operan
–PenguranganDigunakan untuk mengurangi dua operan
-exprKurang UnariDigunakan untuk membalikkan tanda ekspresi
*PerkalianDigunakan untuk mengalikan dua operan
/PembagianDigunakan untuk membagi dua operan
~/DivisiDigunakan untuk membagi dua operan tetapi memberikan output dalam bentuk integer
%ModulusDigunakan untuk memberikan sisa dari dua operan (mengembalikan sisa)

Contoh yang menunjukkan penggunaan semua Operator Aritmatika:

// Dart Program Demonstrating use
// Of all Arithmetic Operators

void main()
{
    int a = 2;
    int b = 3;

    // Adding a and b
    var c = a + b;
    print("Sum  (a + b) = $c");//-> 5

    // Subtracting a and b
    var d = a - b;
    print("Difference (a - b) = $d");//-> -1

    // Using unary minus
    var e = -d;
    print("Negation -(a - b) = $e");//-> 1

    // Multiplication of a and b
    var f = a * b;
    print("Product (a * b) = $f");//-> 6

    // Division of a and b
    var g = b / a;
    print("Division (b / a) = $g");//-> 1.5

    // Using ~/ to divide a and b
    var h = b ~/ a;
    print("Quotient (b ~/ a) = $h");//-> 1

    // Remainder of a and b
    var i = b % a;
    print("Remainder (b % a) = $i");//-> 1
}

2. Operator Relasional​

Kelas operator ini berisi operator-operator yang digunakan untuk melakukan operasi relasional pada operan. Operator-operator tersebut adalah sebagai berikut:

Simbol OperatorNama OperatorDeskripsi Operator
>Lebih besar dariPeriksa operan mana yang lebih besar dan berikan hasil sebagai ekspresi boolean.
<Kurang dariPeriksa operan mana yang lebih kecil dan berikan hasil sebagai ekspresi boolean.
>=Lebih besar dari atau sama denganPeriksa operan mana yang lebih besar atau sama satu sama lain dan berikan hasil boolean
<=Kurang dari atau sama denganPeriksa operan mana yang kurang dari atau sama satu sama lain dan berikan hasil boolean
==Setara denganPeriksa apakah operan sama satu sama lain atau tidak dan berikan hasil boolean.
!=Tidak sama denganPeriksa apakah operan tidak sama satu sama lain dan berikan hasil boolean.

Contoh yang menunjukkan penggunaan Operator Relasional:

// Dart Program Demonstrating use
// Of all Relational Operators
void main()
{
    int a = 2;
    int b = 3;

    // Greater between a and b
    var c = a > b;
    print("a is greater than b (a > b) : $c");//-> false

    // Smaller between a and b
    var d = a < b;
    print("a is smaller than b (a < b) : $d");//-> true

    // Greater than or equal to between a and b
    var e = a >= b;
    print("a is greater than b (a >= b) : $e");//-> false

    // Less than or equal to between a and b
    var f = a <= b;
    print("a is smaller than b (a <= b) : $f");//-> true

    // Equality between a and b
    var g = b == a;
    print("a and b are equal (b == a) : $g");//-> false

    // Unequality between a and b
    var h = b != a;
    print("a and b are not equal (b != a) : $h");//-> true
}

3. Type Test Operators​

Kelas operator ini berisi operator-operator yang digunakan untuk melakukan perbandingan pada operan. Operator-operator tersebut adalah sebagai berikut:

Simbol OperatorNama OperatorDeskripsi Operator
isadalahMemberikan nilai boolean true jika objek memiliki tipe tertentu
is!tidakMemberikan nilai boolean false jika objek memiliki tipe tertentu

Contoh: Menggunakan Operator Uji Tipe dalam program

void main()
{
    String a = 'GFG';
    double b = 3.3;

    // Using is to compare
    print(a is String); //-> true

    // Using is! to compare
    print(b is !int);//-> true
}

as Operator​

Operator as digunakan untuk Typecasting. Operator ini melakukan cast pada saat runtime jika cast valid, jika tidak, akan muncul error. Operator ini terdiri dari dua jenis, yaitu Downcasting dan Type Assertion.

Berikut ini adalah implementasi dari operator as:

void main(){
  // Declaring value
  dynamic value = "Hello";
  
  // TypeCast dynamic -> String
  String str= value as String;
  
  // Print String
  print(str);//-> Hello
}

>>Shift KananMenggeser representasi biner operan ke kanan b bit dan menyisipkan 0 dari kiri.
>>>Shift Kanan tanpa tanda tanganMenggeser representasi biner operan ke kanan b bit (mengabaikan tanda).

5. Operator Penugasan​

Kelas operator ini berisi operator-operator yang digunakan untuk menetapkan nilai pada operan. Operator-operator tersebut berbunyi seperti ini:

Simbol OperatorNama OperatorDeskripsi Operator
=Setara denganDigunakan untuk menetapkan nilai ke ekspresi atau variabel
??=Operator penugasanMenetapkan nilai hanya jika nilainya null

Contoh: Penggunaan Operator Penugasan dalam program

void main()
{
    int a = 5;
    int b = 7;

    // Assigning value to variable c
    var c = a * b;
  
    print("assignment  operator used c = a*b so now c = $c\n");//-> 35

    // Assigning value to variable d
    var d;
  
      // Value is assign as it is null
    d ??= a + b;
  
    print("Assigning value only if d is null");//-> Assigning value only if d is null
    print("d??= a+b so d = $d \n");//-> d = 12
    
    // Again trying to assign value to d
    d ??= a - b;
      // Value is not assign as it is not null
  
    print("Assigning value only if d is null");//-> Assigning value only if d is null
    print("d??= a-b so d = $d");//-> d = 12
    print("As d was not null value was not updated");//-> As d was not null value was not updated
}

Operator Penugasan Gabungan

Selain itu ada cara lain di mana kita dapat menggunakan operator, yaitu operator penugasan majemuk, di mana kita menggabungkan suatu operator dengan operator penugasan lain untuk mempersingkat langkah-langkah dan membuat kode lebih efektif.

6. Operator Logika​

Kelas operator ini berisi operator-operator yang digunakan untuk menggabungkan dua atau lebih kondisi operan secara logis. Operator-operator tersebut berbunyi seperti ini:

Simbol OperatorNama OperatorDeskripsi Operator
&&Dan OperatorDigunakan untuk menambahkan dua kondisi, dan jika keduanya benar maka mengembalikan true.
||Atau OperatorDigunakan untuk menambahkan dua kondisi, dan jika salah satu benar maka mengembalikan true.
!Bukan OperatorDigunakan untuk membalikkan hasil (membuat true menjadi false dan sebaliknya).

Contoh 1: Penggunaan Operator Logika dalam program

void main()
{
    int a = 5;
    int b = 7;  

    // Using And Operator
    bool c = a > 10 && b < 10;
    print(c);//-> false

    // Using Or Operator
    bool d = a > 10 || b < 10;
    print(d);//-> true

    // Using Not Operator
    bool e = !(a > 10);
    print(e);//-> false
}

Contoh 2: (Cara yang Salah)

void main()
{
    int a = 5;
    int b = 7;

    // Using And Operator
    print(a && b);//-> false

    // Using Or Operator
    print(a || b);//-> true

    // Using Not Operator
    print(!a);//-> false
}

Contoh 3: (Cara yang Benar)

void main()
{
    var a = true;
    var b = false;
      // Printing the Values of a & b
    print("a: $a , b: $b\n");//-> a: true , b: false

    // Using And Operator
    print("a && b = ${a&&b}");//-> a && b = false

    // Using Or Operator
    print("a || b = ${a||b}");//-> a || b = true

    // Using Not Operator
    print("!a = ${!a}");//-> !a = false
}

7. Operator Kondisional​

Kelas operator ini berisi operator-operator yang digunakan untuk melakukan perbandingan pada operan. Operator-operator tersebut adalah sebagai berikut:

Simbol OperatorNama OperatorDeskripsi Operator
kondisi ? ekspresi1 : ekspresi2Operator KondisionalVersi sederhana dari pernyataan if-else. Jika kondisinya benar, ekspresi1 akan dieksekusi, jika tidak, ekspresi2 akan dieksekusi.
ekspresi1 ?? ekspresi2Operator KondisionalJika ekspresi1 bukan null, kembalikan nilainya; jika tidak, kembalikan nilai dari ekspresi2.

Contoh: Penggunaan Operator Kondisional dalam program

void main()
{
    int a = 5;

    // Conditional Statement
    var c = (a < 10) ? "Statement is Correct, Geek" : "Statement is Wrong, Geek";
    print(c);//-> Statement is Correct, Geek

    
    // Conditional statement
      int? n;
      // Warning: Operand of null-aware operation '??' has type 'int' which excludes null.
      // For batter practice make both same type to avoid warning
      // var d = n ?? 10;
      var d = n ?? "n has Null value";
      print(d);//-> n has Null value

    // After assigning value to n
    n = 10;
      // we make it all ready null safe 
    //d = n ? ? "n has Null value";
      d = n;
    print(d);//-> 10
}

Keluaran:

Catatan: Dalam Kode Di Atas, Anda Mungkin Melihat Bahwa Variabel 'n' Dinyatakan Sebagai “int? n”. Dengan mendeklarasikan n sebagai int?, Anda menunjukkan bahwa variabel n dapat menyimpan nilai integer atau nilai null. Tanda ? menunjukkan bahwa variabel tersebut dapat berupa null, artinya variabel tersebut dapat diberi nilai null selain nilai integer.

8. Operator Notasi Kaskade:​

Kelas operator ini memungkinkan Anda untuk melakukan serangkaian operasi pada elemen yang sama. Kelas ini memungkinkan Anda untuk melakukan beberapa metode pada objek yang sama. Berikut ini adalah perintahnya:

Simbol OperatorNama OperatorDeskripsi Operator
..Metode BerjenjangDigunakan untuk melakukan beberapa metode pada objek yang sama.
..?Kaskade Korsleting NolDigunakan ketika kita yakin bahwa tidak ada operasi kaskade yang dilakukan pada elemen Null.

Contoh: Penggunaan Operator Notasi Cascade dalam program

// Class 
class GFG {
    var? a;
    var? b;

    void set(x, y)
    {
        this.a = x;
        this.b = y;
    }

    void add()
    {
        var z = this.a + this.b;
        print(z);
    }
}

void main()
{
    // Creating objects of class GFG
    GFG geek1 = new GFG();
    GFG geek2 = new GFG();

    // Without using Cascade Notation
    geek1.set(1, 2);
    geek1.add();//-> 3

    // Using Cascade Notation
    geek2..set(3, 4)
         ..add();//-> 7
}
